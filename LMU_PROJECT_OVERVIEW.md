# LMU Telemetry Lab 專案概覽

這份文件紀錄了 LMU Telemetry Lab 的核心功能特色以及開發過程中的關鍵里程碑。

## 🚀 核心功能亮點

### 1. 極致數據引擎 (DuckDB + FastAPI)
- **高性能分析**：利用 **DuckDB** 的列式存儲與向量化查詢技術，實現海量遙測數據的瞬時檢索。
- **低延遲通訊**：前端 React 與後端 **FastAPI** 透過高效的 JSON/Binary 協議進行通訊，動態圖表與視域切換零延遲。

### 2. 次世代視覺化賽道圖 (Next-Gen TrackMap System)
- **動態追蹤與投影**：支援毫米級精度的 GPS 到投影座標轉換，準確繪製賽車走線。
- **智慧視角切換**：具備 **Static (固定)**、**Follow (跟隨)** 與 **Heading Up (車頭朝上自旋)** 三種專業模式。
- **全視域 MiniMap**：在主視圖底部提供縮小版全場圖，即時掌握整體動態。

### 3. F1 專業級分析介面
- **Telemetry Sync**：多頻道遙測圖表（油門、煞車、速度、RPM）全局游標同步。
- **Delta-T 動態分析**：實時計算與參考單圈 (Reference Lap) 之間的時間盈虧。
- **擬真硬體模擬**：內建數位方向盤 (Steering Lock)、輪胎溫度/壓力熱圖與油耗監測器。

### 4. 頂級視覺體驗 (Glassmorphism UI)
- **玻璃擬態設計**：完美結合 Backdrop Blur 與半透明元素，打造極具科技感的專業儀表板。
- **全局統一提示音 & Tooltips**：自定義玻璃質感提示系統，提供完整的 UX 指引。

### 第二階段：時效連動 (Temporal & Spatial Engine)

- 成功導入「跨賽事參考瀏覽器」，允許玩家從不同的 DuckDB 數據源抓取參考單圈。
- 解決不同車型 (GT/LMH) 之間的數據歸一化比對。

### 第五階段：細節與閉環 (Polishing & Deployment)
